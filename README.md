This project is about online bakery service.
1) Home page.
Home page contains a brief view on menu and buttons like Todays specials that becomes a lightbox when you press it
2) Gallery
Gallery contains lightboxes that can be switched with arrows. It aslo has a filter according to the rype of pastry
3) Menu
Menu has photos of pastry and has a price table. using nth child the table rows highlited. It has filter and search bar

Manage is a managing tables where user can edit, add, read and delete positions 
5) Contact form
Contact form contains validation system. It also checks the password if its strong enough and checks it on equality
